# Hotel-cancellation-Analysis
Hotel booking cancellations are a major challenge in the hospitality industry because they cause revenue loss and inefficient room utilization. This project analyzes hotel booking data to identify patterns and factors that influence cancellations and provide insights that can help hotels reduce cancellation rates.

The dataset used in this project contains 119,390 hotel booking records for City Hotels and Resort Hotels collected between July 2015 and August 2017, including both completed and canceled bookings.

Using Python data analysis and visualization tools, this project explores trends in bookings, cancellation behavior, pricing, and seasonal demand.
# Objectives
- The main objectives of this project are:
- Perform data cleaning and preprocessing
- Conduct Exploratory Data Analysis (EDA)
- Analyze hotel booking trends
- Identify factors affecting cancellations
- Visualize insights using charts and graphs
- Provide business insights and recommendations

# Dataset Information

# The dataset includes booking information such as:
- Hotel type (City Hotel / Resort Hotel)
- Booking status (Canceled or Not)
- Lead time (days between booking and arrival)
- Number of guests (adults, children, babies)
- Market segment and distribution channel
- Room type and customer type
- Average Daily Rate (ADR)
-Special requests and parking spaces

# Tools & Technologies
The following tools and libraries are used:
- Python
- Pandas : Data manipulation
- NumPy  : Numerical operations
- Matplotlib : Data visualization
- Seaborn : Statistical visualization
- Jupyter Notebook

# Key Analysis Performed
- Some important analyses performed in this project include:
- Cancellation rate comparison between City Hotels and Resort Hotels
- Impact of lead time on booking cancellations
- Relationship between Average Daily Rate (ADR) and cancellations
- Seasonal trends in hotel bookings
- Influence of market segment and distribution channel
- Monthly booking trends and demand patterns

# Key Insights
Some insights derived from the analysis:
- City hotels often have higher cancellation rates than resort hotels.
- Longer lead times tend to increase the probability of cancellation.
- Certain market segments and booking channels show higher cancellation behavior.
- Seasonal trends significantly affect hotel demand.
